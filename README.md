# 🏠 Rent Room CLI

>  A command-line room rental management system – written in Go

## 🚀 How to Run

```bash
git clone https://github.com/iamhaghighi/Rent-Room-Mini-Project-CLI.git
cd Rent-Room-Mini-Project-CLI
go run main.go
```


## 📁 Project Structure
```Rent-Room-Mini-Project-CLI/
├── main.go          # Main entry point and CLI menu
├── go.mod           # Go module definition
├── go.sum           # Dependency checksums
└── README.md
```

## ✨ Features
- Add new rooms
- List all rooms
- Rent a room
